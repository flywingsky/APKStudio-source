APK Studio is a open-source and cross-platform IDE for reverse-engineering prebuilt Android applications. Written using Qt framework, it is collaboratively developed by DGenius Xpress along with other contributor(s).

The main aim of this project is to provide ROM makers with the ease of GUI for customization of pre-packaged applications and Android system. The source code for this application is hosted at DGX Code.
